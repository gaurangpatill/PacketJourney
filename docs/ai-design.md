# AI design

AI is not a network sensor. Deterministic tools collect protocol facts and deterministic rules handle directly answerable conditions. The model receives a restricted tool catalog and normalized evidence, then returns validated JSON containing concise explanations, ranked findings, evidence IDs, uncertainty, follow-up questions, and graph emphasis instructions.

Responses referencing missing evidence, returning invalid schemas, or presenting inferred data as verified are rejected. Raw model text is never rendered directly. AI Gateway provides routing, observability, limits, and model configuration; it does not relax the evidence contract.
